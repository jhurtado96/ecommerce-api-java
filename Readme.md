# Ecommerce Prices API

## Descripción

Aplicación Spring Boot para gestionar y consultar precios de productos en una tienda de comercio electrónico.

## Tecnologías Utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Lombok
- RestAssured (Pruebas)
- SwaggerUI

## Requisitos

- JDK 17 o superior
- Maven
